The goal of this is to standardize how we go about tracking engagement and revenue share from our internal marketing efforts. Going forward, it should be second nature to use this tool before posting public facing links with our content!

A UTM, in its simplest form, is just a snipppit of code that we add to the end of a link. This addition lets us know where traffic is coming from and helps better understand the effectiveness of a campaign in the process.

In practice, it’s a summation of 4 parts that lets us group traffic in an easy to understand way, so we'll go over them below.

This paramter (utm_source) definces the source of traffic. This can be a website name, search engine, newletter name, or social network.

The ones we would generally use are listed in a drop down, but if there are others that should be added, let me know!

This parameter (utm_medium) is used to identify the medium used to share and access a link. This will only ever be email, social, or cpc (for ads).

This parameter (utm_campaign) is used to identify the campaign we are running (or in socials case, the platform we are posting from).

In Crafts Group we often refer to this as the Media ID, because this is the part of the UTM code that is tied to an order in Response!

Emails need a unique BE code, but otherwise for socials, it is best practice to use a consistent campagin name! If you feel that there should be some sort of diferentiation, make sure it’s entered in Response!

Note: It’s very important that this be entered into Response before a link is sent out! If you are unsure how to check, or how to do this in the first place, please reach out via teams to Michael or Ben! Otherwise our internal order tables won’t know where it came from, and it will be a bit harder to group revenue generated by it.

This parameter (utm_content) is used to determine what someone clicked to get to our site when there are multiple links pointing to the same url/campaign.

In practice this would be:

Emails: The specific link inside of an email. (shop-now, specific_sale_link, etc.)

Podcasts: The episode number. (ep_1, ep2, etc.)

Youtube: The video title, or a summary. (hand_painting, tubular_cast, etc.)

Socials: The specific sale or campaign that we are promoting. (summer-yarn-sale, etc.)
